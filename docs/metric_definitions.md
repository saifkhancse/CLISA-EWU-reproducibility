# Metric definitions

## balanced accuracy (AccB)

balanced accuracy (AccB) is the average of class-wise recall values. It is used to reduce bias when class frequencies are not perfectly balanced.

In this repository, any table column labelled balanced accuracy (AccB) refers to balanced accuracy.

## Accuracy

Accuracy is the proportion of correctly classified samples among all evaluated samples.

## Macro-F1

Macro-F1 is the unweighted average of per-class F1 scores.
