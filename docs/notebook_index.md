# Provenance notebooks

This folder contains the executed notebooks that directly support the paper results and tables.

The main experiment notebook is `05_clisa_ewu_main_experiments.ipynb`.
The final table-verification notebook is `07_final_results_rebuild_verified.ipynb`.

| Notebook | Purpose |
|---|---|
| `notebooks/provenance/01_seediv_analysis.ipynb` | SEED-IV data and feature analysis used for dataset characterization. |
| `notebooks/provenance/02_faced_domain_gap_analysis.ipynb` | FACED domain-gap analysis used for cross-dataset discussion. |
| `notebooks/provenance/03_dance_reproduction_check.ipynb` | DANCE reproduction check used for comparison/provenance. |
| `notebooks/provenance/04_classical_baselines.ipynb` | Classical machine-learning baseline experiments. |
| `notebooks/provenance/05_clisa_ewu_main_experiments.ipynb` | Main CLISA-EWU and deep-model experiment notebook. |
| `notebooks/provenance/06_final_statistics_and_figures.ipynb` | Final statistics and figure-generation notebook. |
| `notebooks/provenance/07_final_results_rebuild_verified.ipynb` | Final result-table rebuild and verification notebook. |
